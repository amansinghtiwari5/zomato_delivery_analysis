# Zomato Delivery Performance Analytics

## Objective
Analyze food delivery data to identify factors affecting delivery time and operational performance.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- MySQL
- Tableau

## Dataset
Zomato Delivery Operations Dataset (Kaggle)

## Key Steps
1. Data cleaning in Python and MySQL
2. Handling real-world issues (NaN vs NULL, SQL errors)
3. Exploratory Data Analysis
4. SQL business queries
5. Interactive Tableau dashboard

## Key Insights
- Traffic density significantly increases delivery time
- Festival days show higher delivery delays
- Certain cities consistently underperform
- Peak delays occur during evening hours
- Higher-rated delivery partners perform better
